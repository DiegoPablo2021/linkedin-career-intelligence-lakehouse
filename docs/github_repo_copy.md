## Repository Description

Case de engenharia de dados e analytics que transforma exportações do LinkedIn em um lakehouse local com Python, DuckDB, dbt e Streamlit.

## About

LinkedIn export to analytics app with Python, DuckDB, dbt and Streamlit.

## Short Pitch

Projeto end-to-end que recebe múltiplos CSVs do export do LinkedIn, padroniza a ingestão, modela dados analíticos com dbt e entrega um app Streamlit com KPIs, perguntas de negócio e leitura executiva da trajetória profissional.

- camadas `staging`, `intermediate` e `marts` com dbt
